A collection of AI agent skills built for **Claude Code**, **Antigravity**, **OpenCode**, **Cline**, or any skill-supported autonomous agent.

These skills extend your agent's capabilities with specialized tools and workflows, from security auditing to code generation.

You can add skills to your agent using the `npx skills` command.

**Installation:**

```bash
npx skills add https://github.com/shantoislamdev/Skills --skill <skill-name>
```
